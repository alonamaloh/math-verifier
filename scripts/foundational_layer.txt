# foundational_layer.txt — the "assembly zone"
#
# Machine-readable manifest of every library file PERMITTED to use raw CIC
# vocabulary (Quotient.*, unfold, explicit motives, Sort/universe handling,
# congruenceOf/transport_proposition, …). Consumed by scripts/cic_leak_report;
# rationale and policy live in docs/foundational-layer.md.
#
# Syntax:
#   - one path per line, relative to the repo root
#   - a trailing '/' means "this directory and everything under it"
#   - '#' starts a comment; blank lines are ignored
#
# Everything NOT matched here is "user space" and must stay CIC-free. The
# manifest is meant to be BOUNDED AND SHRINKING: removing a file from this
# list (because its CIC use was lifted into a nicer tool) is progress.
